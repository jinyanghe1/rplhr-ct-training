```
└──────────────────────────────────────────────────────────┘
                            │
                            │ git push
                            ▼
┌──────────────────────────────────────────────────────────┐
│ AutoDL 拉取最新代码                                       │
│ ─────────────────                                         │
│ cd /root/autodl-tmp/RPLHR-CT/code/code                    │
│ git pull origin main                                      │
└──────────────────────────────────────────────────────────┘
                            │
                            ▼
┌──────────────────────────────────────────────────────────┐
│ 启动训练                                                  │
│ ─────────────────                                         │
│ nohup python train.py train \                             │
│     --path_key SRM \                                      │
│     --gpu_idx 0 \                                         │
│     --net_idx TVSRN \                                     │
│     > train.log 2>&1 &                                    │
└──────────────────────────────────────────────────────────┘
                            │
                            ▼
┌──────────────────────────────────────────────────────────┐
│ 监控训练                                                  │
│ ─────────────────                                         │
│ tail -f train.log          # 查看日志                     │
│ watch -n 1 nvidia-smi      # 查看 GPU                     │
│ tmux attach -t training    # 连接会话                     │
└──────────────────────────────────────────────────────────┘
                            │
                            ▼
┌──────────────────────────────────────────────────────────┐
│ 保存结果                                                  │
│ ─────────────────                                         │
│ tar -czf results.tar.gz model/ val_output/                │
│ aliyunpan upload results.tar.gz /RPLHR-CT-Results/        │
└──────────────────────────────────────────────────────────┘
```

## 🔧 常用命令速查

### 本地命令

```bash
# 推送代码
git add . && git commit -m "update" && git push origin main

# 打包数据
bash prepare_data_local.sh ../data ./packaged_data
```

### AutoDL 命令

```bash
# 连接实例
ssh root@region.autodl.com -p 12345

# 登录阿里云盘
aliyunpan login

# 下载数据
bash download_data.sh

# 启动训练
bash start_training.sh

# 查看日志
tail -f /root/autodl-tmp/RPLHR-CT/train.log

# 查看 GPU
nvidia-smi
watch -n 1 nvidia-smi

# 查看进程
ps aux | grep python

# 杀死进程
kill -9 <PID>
```

### Tmux 会话管理

```bash
# 创建会话
tmux new -s training

# 分离会话 (按 Ctrl+B, 然后按 D)

# 重新连接
tmux attach -t training

# 列出会话
tmux ls

# 杀死会话
tmux kill-session -t training
```

## 💰 费用优化建议

1. **按需开关机**: 不训练时关机，只保留数据盘
2. **使用无卡模式**: 数据准备阶段使用无卡模式（更便宜）
3. **数据持久化**: 将数据存储在数据盘，关机不丢失
4. **及时备份**: 重要模型及时上传到阿里云盘

```
训练日程表示例:
┌─────────┬─────────┬─────────────────┐
│ 时间     │ 操作     │ 费用            │
├─────────┼─────────┼─────────────────┤
│ 09:00   │ 开机     │ 开始计费        │
│ 09:10   │ 启动训练 │ GPU满负荷       │
│ 18:00   │ 训练结束 │ 保存模型        │
│ 18:10   │ 关机     │ 停止计费        │
└─────────┴─────────┴─────────────────┘
```
